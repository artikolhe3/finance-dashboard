# 💰 Finance Dashboard

## 📌 Overview

Finance Dashboard is a simple and responsive web application built using React.
It helps users track their **income, expenses, and overall balance** in a clean and visual way.

---

## 🚀 Features

* 📊 Dashboard summary (Income, Expense, Balance)
* 📋 Transactions list
* 🔍 Search & filter functionality
* 👤 Role-based UI (Admin / Viewer)
* 📈 Charts for data visualization
* 💡 Insights section

---

## 🛠️ Tech Stack

* ⚛️ React (Frontend)
* 🎨 Tailwind CSS (Styling)
* 📊 Recharts (Charts)

---

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/artikolhe3/finance-dashboard.git
```

Go to project folder:

```bash
cd finance-dashboard
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
